Project 02

Building a VBA-based ERP System to Digitalize and Streamline Supply Chain Operations

Supply Chain ERP Implementation Automation Business Process Modeling Database Management

Overview

A local electrical-equipment manufacturer ran its operations on disconnected spreadsheets and manual workflows. Orders, procurement, and production were tracked in isolation, with no single source of truth. I designed and built a working ERP system to pull it all together.

  • Unified three core workflows (Order-to-Cash, Procure-to-Pay, and Plan-to-Stock) into one Excel/VBA system
  • Built validated input forms, auto-generated transaction IDs, credit checks, and automatic cost and material-planning calculations
  • Backed it with a structured worksheet database and live inventory tracking
  • Replaced manual, error-prone steps to improve accuracy, traceability, and audit readiness
